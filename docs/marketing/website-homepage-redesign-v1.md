# Runory Website Homepage Redesign v1

## Goal
Transform the website from a technical platform landing page into a commercial product marketing website.

## Positioning
Runory is the field service operating system for the Agent era.

CRM, Sales, FSM, and Omnichannel Intake unified into one adaptive platform for service businesses.

## Homepage Structure

### 1. Hero
Message:
- Field service operating system for the Agent era
- One platform from customer conversation to completed service

Primary CTA:
- Start a Pilot

Visual placeholder:
- Operations Workspace
- Mobile Field Experience
- Agent Interaction

### 2. Business Problem
Three concise problems:
- Disconnected tools
- Manual operations
- Rigid software

### 3. Complete Service Platform
Flow:
Voice / SMS / Web Intake
→ CRM
→ Sales
→ FSM
→ Payment & Retention

### 4. Product Experience
Show real product surfaces:
- Operations Workspace
- Mobile Field App
- Agent Interface

### 5. Why Runory
Key differentiation:
- External Super Agent first
- Governed execution
- Adaptive runtime
- Faster implementation

### 6. Focused Pilot
Message:
- Start with one workflow
- One team
- Mature customers can achieve initial deployment in 1-2 weeks

### Visual Direction
Maintain Runory brand DNA:
- Warm ivory background
- Near black typography
- Orange accent
- Premium whitespace
- Product-focused visuals

Avoid:
- AI gradient style
- Excessive icons
- Technical jargon above the fold
- Dense text blocks
