# Runory demo avatars

These deterministic local SVG assets are generated by:

```sh
pnpm --filter @runory/cloud avatars:generate
```

Generator: DiceBear 9.4.2 with the `notionists-neutral` style. Both the
DiceBear core package and this style are distributed under the MIT license.
The assets are committed so the product never depends on an external avatar
API at runtime.
