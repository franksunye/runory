# Runory demo avatars

These deterministic local SVG assets are generated by:

```sh
pnpm --filter @runory/cloud avatars:generate
```

Generator: DiceBear 9.4.2 with the `notionists-neutral` package (MIT), based on
the Notionists artwork by Zoish (CC0 1.0). The assets are committed so the
product never depends on an external avatar API at runtime.
